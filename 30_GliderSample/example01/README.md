## これは何？

IMU（Madgwick）で推定した姿勢角 `theta`（roll/pitch/yaw）を使って、サーボ角 `ang` を動かす最小例です。

式は概ね次の形です（I/Dなし）：

- `ang = ang0 + K * (theta0 - theta)`（`mode=1` のとき）

## 挙動

- `mode=0`（手動相当）：`ang = ang0` なので、姿勢に関係なくサーボが指定角（オフセット）に行きます
- `mode=1`（自動相当）：姿勢角が `theta0`（目標）からズレると、ズレを打ち消す方向にサーボが動きます（P制御）

## ポイント（学びどころ）

- **閉ループの最小構成**：目標（`theta0`）−計測（`theta`）→ 操作量（`ang`）
- `servo.write(ang + 90)` のように、制御量（±）をサーボ角（0..180）へ写像している
- 角度やゲインの符号が「機体の取り付け向き」に依存する（符号が逆だと発散します）
