# 30_GliderSample

`10_Lesson/` → `20_Exercise/` を終えた後に読むことを想定した、**グライダー/機体制御のサンプル集**です。

## 目的

- IMU（姿勢推定）→ 操作量の計算 → サーボ、という **閉ループ制御の一連の流れ**を読む
- 実機条件（投げ上げ・加速度・ノイズ・サーボ制約など）を意識した **実装上の工夫**を読む
- 自分の課題・実験に合わせて、改造の出発点として使う

## 進め方（推奨）

1. この `README.md` で全体像をつかむ
2. 興味のあるフォルダ（`example01/` など）の `README.md` を読む
3. `.ino` を開き、`setup()` → `loop()` → 制御計算（誤差計算など）の順に追う
4. どこが「教科書通り」で、どこが「実装上の工夫」かを意識して比較する

## ハードウェア前提（共通の注意）

- サーボの電源はUSBだけでは不足する場合があります（外部電源 + GND共通化）
- サーボ角は `0..180` に制限され、ソフト側でも `constrain()` / クリップが必要です
- 姿勢推定（Madgwick）は「静止〜準静的」では強い一方、強い加速度が乗る区間では角度が乱れます
