# 30_GliderSample

`10_Lesson/` → `20_Exercise/` を終えた後に読むことを想定した、**グライダー/機体制御のサンプル集**です。

ここにある各フォルダは、`SampleProgram/` にあるプログラムを **教材用にコピー**したものです（元のプログラム自体は `SampleProgram/` 側）。

## 目的

- `60_...` 系：IMU（姿勢推定）→ サーボ操作量の計算、という**閉ループ制御の流れ**を読む
- `example_tsuchiyama/`：実機条件（投げ上げ・加速度・ノイズ・サーボ制約）を意識した**実装上の工夫**を読む

## 進め方（推奨）

1. 各フォルダの `README.md`（このフォルダ）を先に読む
2. 次に `.ino` を開き、`setup()` → `loop()` → 制御計算（誤差計算など）の順に追う
3. どこが「教科書通り」で、どこが「実装上の工夫」かを意識して比較する

## ハードウェア前提（共通の注意）

- サーボの電源はUSBだけでは不足する場合があります（外部電源 + GND共通化）
- サーボ角は `0..180` に制限され、ソフト側でも `constrain()` / クリップが必要です
- 姿勢推定（Madgwick）は「静止〜準静的」では強い一方、強い加速度が乗る区間では角度が乱れます
